## Descripción del Proyecto
Este proyecto consiste en un sitio web informativo de una sola página que presenta al Grupo SuperMegaPro. El sitio ha sido construido exclusivamente con HTML5 y CSS3, aplicando estructuras semánticas y un diseño basado en tarjetas para mostrar los perfiles de los integrantes.

## Características Técnicas
- **Semántica HTML5:** Uso de etiquetas `<header>`, `<main>`, `<section>`, `<nav>` y `<footer>`.
- **Organización de Estilos:** Uso de un archivo CSS externo (`styles.css`) para separar la estructura de la presentación.
- **Identificadores y Clases:**
  - IDs únicos: `#equipo`, `#miembros`, `#contacto` y `#navegacion-principal`.
  - Clases reutilizables: `.tarjeta-estudiante`, `.texto-destacado`, `.lista-proyectos` y `.menu-lista`.
- **Diseño Visual:** Layout de tarjetas con sombras, bordes redondeados y navegación funcional mediante anclas.
