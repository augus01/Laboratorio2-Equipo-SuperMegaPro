Instalación de Docker Desktop

Sigue este tutorial para instalar Docker Desktop en Windows, ejecutar tu primer contenedor y publicar una página web HTML desde un contenedor.

01

Descargar Docker Desktop

1

Abrir el navegador

Abre tu navegador de preferencia y dirígete a la página oficial de Docker.

URL https://www.docker.com/get-started/
2

Descargar el instalador

Busca el botón "Download Docker Desktop" y descarga el archivo correspondiente a tu sistema operativo Windows.

💡 Asegúrate de seleccionar la versión correcta para tu arquitectura (AMD64 o ARM64).
02

Instalar Docker Desktop

1

Ejecutar el instalador

Busca el archivo descargado en tu carpeta de descargas y ejecútalo con doble clic.

Archivo Docker Desktop Installer.exe
2

Aceptar términos y configurar WSL 2

En el asistente de instalación, acepta los términos de uso. Si aparece la opción de usar WSL 2, déjala activada por defecto, es la configuración recomendada.

💡 WSL 2 (Windows Subsystem for Linux) mejora el rendimiento de Docker en Windows.
3

Completar la instalación

Haz clic en Next e Install hasta que el proceso finalice. El instalador descargará los componentes necesarios automáticamente.

4

Finalizar y reiniciar

Al terminar, haz clic en Close. Si el instalador lo solicita, reinicia el equipo para que los cambios surtan efecto.

⚠️ No omitas el reinicio si es solicitado, puede causar errores al iniciar Docker.
03

Primer arranque y prueba

1

Abrir Docker Desktop

Busca Docker Desktop en el menú Inicio de Windows y ábrelo. La primera vez puede tardar unos segundos en iniciar.

2

Verificar que está en ejecución

Espera hasta que el estado en la interfaz cambie a Running. También verás el ícono de la ballena de Docker activo en la bandeja del sistema.

🐋 El ícono de la ballena en la barra de tareas indica que Docker está corriendo correctamente.
3

Ejecutar el contenedor de prueba

Abre Terminal o PowerShell y ejecuta el siguiente comando para verificar que Docker funciona correctamente.

CMD / PowerShell docker run hello-world
✓

Resultado esperado

La primera vez, Docker descargará la imagen hello-world desde Docker Hub. Luego mostrará un mensaje confirmando que la instalación fue exitosa y que Docker se ejecutó correctamente.

✅ Si ves el mensaje "Hello from Docker!", la instalación está completa y funcional.
04

Primer contenedor web

1

Crear la carpeta del proyecto

Crea una carpeta en tu equipo para el proyecto y dentro de ella crea un archivo index.html con tu página web.

Ruta ejemplo C:\docker-lab-web\index.html
2

Crear el Dockerfile

En la misma carpeta crea un archivo llamado Dockerfile sin extensión con el siguiente contenido. Usa nginx como servidor web.

Dockerfile FROM nginx:alpine COPY index.html /usr/share/nginx/html/index.html
3

Construir la imagen

Abre PowerShell o CMD, navega hasta la carpeta del proyecto y construye la imagen Docker.

PowerShell docker build -t mi-web-html .
💡 El punto al final indica que el Dockerfile está en la carpeta actual.
4

Ejecutar el contenedor

Levanta el contenedor publicando el puerto 80 al 8080 de tu máquina y accede desde el navegador.

PowerShell docker run -d -p 8080:80 --name mi-web-container mi-web-html
✓

Verificar en el navegador

Abre tu navegador y accede a la siguiente dirección. Deberías ver tu página HTML servida desde el contenedor Docker.

Navegador http://localhost:8080
🛑 Para detener: docker stop mi-web-container — Para eliminar: docker rm mi-web-container
